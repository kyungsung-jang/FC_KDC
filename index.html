이미 진행 중이신가요?

STEP 1

내일배움카드 신청

HRD-Net(고용보험 직업훈련 포털)에서 내일배움카드를 온라인으로 신청합니다.

  1. 1

    HRD-Net 접속 및 회원가입

    HRD-Net 공식 사이트에 접속하여 회원가입을 완료합니다. 이미 계정이 있다면 로그인하세요.

    • ✔주민등록번호 또는 외국인등록번호 필요
    • ✔본인 명의 휴대폰 번호로 인증
    • ✔이메일 주소 등록 (결과 알림 수신)
    HRD-Net 메인 화면
    🌐 HRD-Net 바로가기
  2. 2

    내일배움카드 신청 메뉴 찾기

    로그인 후 메뉴에서 내일배움카드 신청 페이지로 이동합니다.

    • ✔상단 메뉴 [직업 능력 개발] 클릭
    • ✔[국민내일배움카드] 선택
    • ✔[발급 신청] 메뉴 클릭
    메뉴 위치 안내
  3. 3

    신청서 작성 및 제출

    신청서 양식에 따라 정보를 입력하고 제출합니다.

    • ✔고용보험 가입 여부 확인 (직장인 / 구직자 선택)
    • ✔훈련 희망 분야 입력
    • ✔카드 수령 주소 정확히 기재
    • ✔신청 완료 후 문자 알림 수신 확인
    신청서 작성 화면
    ℹ️

    신청 후 심사 기간 약 5영업일이 소요됩니다. 심사 결과는 문자와 이메일로 안내됩니다.

카드 신청이 완료되었나요?

STEP 2

카드 발급 및 등록

카드가 집으로 배송되면 HRD-Net에서 카드를 등록합니다.

  1. 1

    카드 수령 확인

    심사 승인 후 내일배움카드가 등록 주소로 우편 배송됩니다.

    • ✔배송 기간: 승인 후 약 7~14일 소요
    • ✔카드 봉투 및 안내문 함께 확인
    • ✔카드 뒷면 서명 필수
    📱

    모바일 신청의 경우, 선택하신 은행에서 안내되는 '내일배움카드' 발급 안내 문자 수령 후 은행 카드 앱을 통해 발급 신청해주세요.

    ⚠️

    카드 수령 전에는 수강신청이 불가합니다. 배송이 지연될 경우 고용센터(1350)로 문의하세요.

  2. 2

    국민내일배움카드 등록 확인

    카드를 받은 후 고용24에서 카드 정보를 확인합니다.

    • ✔고용24 로그인 → 마이페이지
    • ✔[직업훈련관리] 선택
    • ✔[국민내일배움카드] 선택
    • ✔[카드사용내역] 메뉴에서 카드 정보 확인
    • ✔발급받은 카드 번호와 일치하는지 확인
    카드사용내역 확인 화면
    🌐 고용24 바로가기

카드 등록이 완료되었나요?

STEP 3

수강신청 및 결제

수강신청 방법은 두 가지 루트가 있습니다. 본인 상황에 맞는 루트를 선택해주세요.

  1. 1

    고용24에서 과정 검색

    고용24(구 HRD-Net) 훈련과정 검색에서 수강할 과정을 찾습니다.

    • ✔상단 메뉴 훈련과정 검색 클릭
    • ✔검색창에 패스트캠퍼스 또는 과정명 입력
    • ✔지역, 훈련유형, 기간 필터 활용
    고용24 훈련과정 검색 화면
    🔍 고용24 바로가기
  2. 2

    수강신청

    원하는 과정을 선택하여 수강신청을 합니다.

    • ✔과정 상세 페이지에서 수강신청 버튼 클릭
    • ✔훈련 시작일·종료일 확인
    • ✔훈련기관 약관 동의
    수강신청 화면
  3. 3

    내일배움카드로 결제

    수강신청 후 내일배움카드로 자기부담금을 결제합니다.

    • ✔결제 방법에서 내일배움카드 선택
    • ✔자기부담금 확인 (훈련비의 일부, 보통 20~55%)
    • ✔간편인증으로 본인 확인 후 결제 완료
    결제 화면
    🎉

    결제 완료! 훈련 시작일에 맞춰 출석하면 됩니다.

  1. 1

    패스트캠퍼스 홈페이지에서 과정 검색

    패스트캠퍼스 공식 홈페이지에서 수강할 과정을 검색하고 선택합니다.

    • ✔패스트캠퍼스 홈페이지 접속 후 로그인
    • ✔검색창에서 원하는 과정명 검색
    • ✔과정 상세 페이지에서 내용 및 일정 확인
    패스트캠퍼스 과정 검색 화면
    🎓 패스트캠퍼스 바로가기
  2. 2

    패스트캠퍼스에서 결제

    과정 상세 페이지에서 내일배움카드 결제를 진행합니다.

    • ✔과정 페이지 내 국비 신청하기 버튼 클릭
    • ✔내일배움카드 번호 입력 및 자기부담금 확인
    • ✔간편인증으로 본인 확인 후 결제 완료
    국비 결제 화면
    ⚠️

    결제 후 고용24에서 수강신청을 반드시 완료해야 수강이 확정됩니다.

  3. 3

    고용24에서 수강신청 완료

    패스트캠퍼스 결제 후 고용24에서 수강신청을 추가로 진행해야 최종 확정됩니다.

    • ✔고용24 접속 후 로그인
    • ✔신청한 과정 검색 후 수강신청 클릭
    • ✔수강신청 완료 문자 수신 확인
    고용24 수강신청 화면
    🖥️ 고용24 바로가기
    🎉

    수강신청 완료! 훈련 시작일에 맞춰 출석하면 됩니다.

🎊
